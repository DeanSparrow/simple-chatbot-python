# BNU-BOT
A simple chatbot build with Python, NLTK & SKLearn.

## Installation

Installation is simplest:

1. Clone the repository
2. Install dependencies with `pip install -r requirements.txt`
3. You are good to go!    

#### Installation notes

BNU-BOT depends on:
 1. `NLTK`
 2. `SKLearn`
